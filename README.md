# ELK-Track

가상 환경 실행
```
python -m venv venv
source venv/Scripts/activate 
```

파일 구조
```
fastapi_project/
│
├── app/
│   ├── __init__.py     
│   ├── main.py          
│   ├── models.py        # 데이터베이스 모델
│   ├── routes.py        # API 엔드포인트
│   ├── schemas.py       # Pydantic 스키마
│
├── requirements.txt     # 패키지 의존성 관리
```
